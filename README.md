# Arquitectura Web de 3 Capas en AWS

Proyecto de portfolio que implementa una arquitectura web clásica de 3 capas (presentación, aplicación y datos) en AWS, construida manualmente desde la consola/CLI como primer paso antes de automatizarla con Terraform.

## Objetivo

Demostrar el despliegue de una infraestructura web productiva y segura en AWS, aplicando buenas prácticas de:
- Segmentación de red (VPC, subredes públicas/privadas)
- Alta disponibilidad (múltiples zonas de disponibilidad)
- Seguridad por capas (Security Groups, NACLs)
- Separación de responsabilidades entre capas
